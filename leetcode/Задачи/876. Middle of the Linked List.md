### Ссылка:
[876. Middle of the Linked List](https://leetcode.com/problems/middle-of-the-linked-list/)

### Описание:
Дан односвязный список, верните средний узел связанного списка

Если есть два средних узла, верните второй средний узел

### Объяснение:
Если начать рассматривать задачу с минимально возможного списка и постепенно его увеличивать, можно заметить, что середина списка "обновляется" через каждые два промежутка
![[Pasted image 20251120180223.png]]
Это позволяет сделать вывод, что длина увеличивается быстрее, чем его середина. Тогда, нужно добавить два указателя, быстрый будет искать конец связного списка и проходить быстрее, тогда как медленный указатель будет искать середину. 
Мой изначальный код:
```php
/**
 * Definition for a singly-linked list.
 * class ListNode {
 *     public $val = 0;
 *     public $next = null;
 *     function __construct($val = 0, $next = null) {
 *         $this->val = $val;
 *         $this->next = $next;
 *     }
 * }
 */
class Solution {

    /**
     * @param ListNode $head
     * @return ListNode
     */
    function middleNode($head) {
        $tmpHead = $head;
        $len = 1;
        while ($tmpHead->next) {
            $len++;
            $tmpHead = $tmpHead->next;
        }
        $halfLen = (int) floor($len / 2);
        while ($halfLen !== 0) {
            $halfLen--;
            $head = $head->next;
        }
        return $head;
    }
}
```

Переписанный код, после того как изучил быстрый и медленный указатель:
```php
/**
 * Definition for a singly-linked list.
 * class ListNode {
 *     public $val = 0;
 *     public $next = null;
 *     function __construct($val = 0, $next = null) {
 *         $this->val = $val;
 *         $this->next = $next;
 *     }
 * }
 */
class Solution {

    /**
     * @param ListNode $head
     * @return ListNode
     */
    function middleNode($head) {
        $fast = $head;
        $slow = $head;
        while ($fast->next) {
            $slow = $slow->next;
            $fast = $fast->next->next;
        }
        return $slow;
    }
}
```
